# nate_style

natepiano's style guidelines for rust/bevy & ai prompts

## Structure

- `prompt/` - Individual rule files covering clarity, interactions, philosophy, structure, token use, variable handling, workflow and well, prompts.
- `rust/` - Individual rule files covering imports, lints, patterns, visibility, project setup, and bevy-specific conventions.
- `*.base` - [Obsidian Bases](https://help.obsidian.md/bases) views that filter rules by tag. For Humans.
