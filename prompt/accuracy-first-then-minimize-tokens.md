## Accuracy first, then minimize tokens

Do the most correct and accurate work first. Then, among equally correct approaches, prefer the one that uses fewer tokens. Never sacrifice accuracy for brevity. A few extra input tokens to prevent output errors is a good tradeoff.

Strategies: tagged section reuse, shared workflows, template variables, script delegation, verbosity elimination, pattern consistency.
