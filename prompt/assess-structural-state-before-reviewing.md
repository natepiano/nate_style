## Assess structural state before reviewing

Before critiquing a command, determine its current structural state. A pattern-compliant command (already uses tagged sections, keyword formats, interaction patterns) needs incremental improvements. A pattern-migration command (has structure but different conventions) needs restructuring first. Don't nitpick details on a command that needs fundamental reorganization.
